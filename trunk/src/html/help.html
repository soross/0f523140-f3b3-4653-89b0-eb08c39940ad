1. 微博招聘网是什么？
微博招聘网(www.weiboZP.com)是一款全新的微博招聘求职聚合平台。可以同时绑定多个主流微博，一次发布职位同时推送。 消息同步，转发，评论，收藏，分析一切尽在轻松掌握。

2. 支持的微博平台
目前微博招聘网支持主流的微博平台，如新浪、搜狐、网易等。随着接口的开放，我们支持的微博也会越来越多。

3. 如何关联其他主流博客？
你只需花费30秒进行注册就可以登陆，登陆后通过关联你的多个微博帐号，就可以实时进行一键同步更新！

4. 目前微博开放的绑定协议？
Basic Authentication 和 Oauth协议。

5. 账号安全与保密
对于OAuth协议，我们只是关联到您的微博网站进行相应的认证，对于Basic认证，采用加密的方式，不会对您的账号安全造成任何威胁。

6. 如何联系你们呢？
